## MEV

MEV是在生成区块过程中通过重新排序交易、穿插或审查等交易而提取的总价值。

## 三明治套利

观察内存池，在一笔交易完成之前买入，在完成之后卖出，以完成套利。

### 步骤

1. 添加x-ETH LP
2. 监控内存池的x-ETH交易
3. 一旦有导致滑点10%的买单交易，则发起一笔买单交易
4. 交易成功之后，则发起一笔卖单交易
5. 记录套利日志

## 清算套利

借闪电贷，清算质押物，还闪电贷

### 步骤

1. 监控质押物的价格
2. 一旦质押物的价格等于清算价格，则发起一笔闪电贷交易，借入一笔钱，帮系统偿还债务，卖出质押物，还掉这一笔钱
